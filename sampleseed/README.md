# sampleseed

Modern sample sales-focused redesign for SEED Homes, based only on public content from `www.seed-homes.com`.

The page keeps the current live-site flow and links:

- Buy Now: `https://seed-homes.myshopify.com`
- More Info / Register: Google Form from the live site
- Virtual tours: CloudPano links from the live site
- Videos: YouTube links from the live site

Only public live-site content is used in this sample.
